# 文件下载程序

这是一个支持多语言的文件下载程序，具有进度显示、错误处理和 MD5 校验功能。

## 特性

- 支持通过 URL 下载文件
- 显示下载进度
- 支持取消下载
- 自动重试功能
- 支持文件的 MD5 校验
- 中英文消息支持

## 安装

确保您的环境中安装了 Python 3.x 和 `requests` 库。可以使用以下命令安装所需的库：

```bash
pip install requests tqdm
```
## 使用
可以通过命令行运行程序，传入要下载的文件的 URL：
```bash
python download.py <url>
```
